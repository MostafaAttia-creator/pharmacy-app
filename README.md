Pharmacy Management System
Project Overview

This project is a simple Pharmacy Management System built to manage medicines, sales, and customer data efficiently.

The application allows users to:

View available medicines
Manage stock quantities
Record sales transactions
Handle customer purchases

The goal of this project is to simulate a real-world pharmacy workflow in a simple and interactive way.

Technologies Used
Python
Streamlit
SQLite
Pandas
Features
Add and manage medicines
Update stock automatically after sales
Record customer purchases
Simple and interactive user interface
Lightweight database using SQLite
Project Structure
app.py → Main application
pharmacy.db → Database file
schema.sql → Database schema
Other supporting files
How to Run

Clone the repository:
git clone https://github.com/MostafaAttia-creator/pharmacy-app.git

Navigate to the project folder:
cd pharmacy-app

Install dependencies:
pip install -r requirements.txt

Run the application:
streamlit run app.py

Usage
Select a medicine from the list
Enter quantity
Confirm purchase
Stock will be updated automatically
